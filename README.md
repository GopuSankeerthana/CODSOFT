CodSoft Data Science Internship

Overview
This repository contains the tasks completed as part of the CodSoft Data Science Internship. The projects demonstrate data analysis, machine learning, data preprocessing, model training, and prediction techniques using Python and popular data science libraries.

Internship Tasks

Task 1: Titanic Survival Prediction
Objective: Predict whether a passenger survived the Titanic disaster using machine learning.
Technologies Used:
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

Files:
* Titanic-Dataset.csv
* task1_titanicds_prediction.ipynb
* README.md

Key Concepts:
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Classification Models
* Model Evaluation
