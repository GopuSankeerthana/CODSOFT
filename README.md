# 📊 CodSoft Data Science Internship

## 📌 Overview

This repository contains the tasks completed during the **CodSoft Data Science Internship**. The projects focus on machine learning, data preprocessing, data analysis, model training, prediction, and evaluation using Python and popular data science libraries.

The internship tasks provide practical experience in applying data science concepts to real-world datasets.

---

## 🎯 Internship Tasks

### Task 1: Titanic Survival Prediction

**Objective:**

Build a machine learning model to predict whether a passenger survived the Titanic disaster using passenger details.

**Key Activities:**

* Data cleaning and preprocessing
* Handling missing values
* Feature selection
* Classification model training
* Prediction and evaluation

**Files:**

* Titanic-Dataset.csv
* task1_titanicds_prediction.ipynb
* README.md

---

### Task 2: Movie Rating Prediction

**Objective:**

Develop a machine learning model to predict movie ratings based on attributes such as genre, director, actors, and other movie characteristics.

**Key Activities:**

* Data preprocessing
* Feature encoding
* Regression model training
* Movie rating prediction
* Performance evaluation

**Files:**

* IMDb movie dataset.csv
* Task2_MoviePrediction.ipynb
* README.md

---

### Task 3: Credit Card Fraud Detection

**Objective:**

Build a machine learning model to identify fraudulent credit card transactions using transaction-related features.

**Key Activities:**

* Data preprocessing
* Handling class imbalance
* Feature analysis
* Classification model implementation
* Fraud prediction and evaluation

**Files:**

* creditcard.csv
* task3_CreditCardFraudDetection.ipynb
* README.md

---

### Task 4: Iris Flower Classification

**Objective:**

Build a classification model to predict the species of Iris flowers based on sepal and petal measurements.

**Key Activities:**

* Dataset exploration
* Data preprocessing
* Classification model training
* Flower species prediction
* Model evaluation

**Files:**

* IRIS.csv
* Iris_Classification.ipynb
* README.md
