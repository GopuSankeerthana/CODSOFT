# 🌸 Iris Flower Classification Using Machine Learning

## 📌 Project Overview

The Iris Flower Classification project is a machine learning application designed to classify Iris flowers into their respective species based on their physical characteristics. The model analyzes flower measurements and predicts whether the flower belongs to the Setosa, Versicolor, or Virginica species.

This project demonstrates the complete machine learning workflow, including data preprocessing, model training, prediction, and evaluation.

---

## 🎯 Objective

The objective of this project is to build a classification model capable of accurately identifying Iris flower species using sepal and petal measurements.

---

## 📂 Dataset Information

Dataset Name: Iris Dataset

The dataset contains measurements of Iris flowers and their corresponding species labels.

### Input Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target Variable

* Species

### Classes

* Iris Setosa
* Iris Versicolor
* Iris Virginica
